# Introduction

```90 min```

Our programming language of choice for this course will by [Python](https://www.python.org/). Prior experience with coding is not required to understand this course as we will be going over the basics of Python in this first lesson.

If you've never programmed before then this course will teach you what you need to know. But this means that you will ultimately _teach yourself_ Python and that can seem really daunting! After all, learning a programming language is a lot like learning a language: there is syntax, grammar and vocabulary that you will need to internalize if you want to speak it fluently. This takes **time** and **sustained effort**. However, luckily for us, Python is among the easiest languages to learn.

If you've ever browsed through any computer program, you may have noticed that the design and purpose of the program is nearly completely opaque. For many popular programming languages like Java and C, it is rarely easy to tell what a program does by looking at it with no prior context or understanding. On the other hand, Python has a 20+ year development history that emphasises readability. It should be clear, without much mental effort, to look at Python code and understand what it does.

To that end, I hope you will find learning Python to be straightforward.


## Objectives

This lesson will teach you to:

- Be able to perform simple arithmetic in Python.
- Use the `print()` function.
- Use the `if`, `elif` and `else` statements.
- Create `for` and `while` loops, and understanding the `continue` and `break` statements.
- Iterate through lists and dictionaries.
- Write list and dictionary comprehensions.
- Write functions that solve problems.
